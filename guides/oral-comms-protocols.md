## Overview: Communication Under Pressure

In an emergency—fire, attack, medical crisis—communication must be instant and perfectly clear. A misheard word can result in death. "Move to the west wall" heard as "Move to the wet hall" sends people to the wrong place. Under stress, people speak unclearly, hear poorly, and misunderstand easily.

This guide covers practical techniques for ensuring messages are transmitted accurately and received correctly, even when conditions are poor:

1. **Clear speech principles** (how to speak so you're understood)
2. **Standardized formats** (structured messaging reduces error)
3. **Radio procedure** (discipline for radio communication)
4. **Hand signals** (tactical movement without voice)
5. **Stress effects** (what happens to communication under pressure)

</section>

<section id="clear-speech">

## Clear Speech Principles

Speaking clearly is skill that can be learned and practiced.

### Enunciation & Pace

**Enunciation:**
- Speak from the diaphragm (deep breaths; project voice)
- Pronounce every consonant distinctly (don't slur "th" sounds)
- Hold vowel sounds slightly longer than normal (allows listener to process)
- Avoid rushing; speak at 70% normal pace

**Examples of poor vs. good speech:**

**Poor:** "thers'a fir in sector twuh, move west"
- Slurred consonants, run-on words, mumbled location

**Good:** "There is a fire in Sector Two. Move... west."
- Distinct pronunciation, pauses between phrases, clearly spoken location

### Brevity with Clarity

- **Don't ramble:** Every word should add information
- **Front-load critical information:** "FIRE in Sector Two" (the critical part first)
- **Use short sentences:** Less chance of error mid-message
- **Repeat key information:** Say location/instruction twice if critical

### Adaptive Delivery

Adjust speech based on listener and environment:

- **Shouting:** In loud environment (combat, machinery noise), speak louder and slower
- **Whisper:** In stealth situation, speak very quietly but still clearly (not hissed)
- **Formal:** In diplomatic/official contexts, use careful articulation
- **Casual:** In low-stress, known audience, can relax somewhat

## Phonetic Alphabet

When spelling is critical (especially over radio where reception is imperfect):

### NATO Phonetic Alphabet

| Letter | Phonetic | | Letter | Phonetic |
|--------|----------|---|--------|----------|
| A | Alpha | | N | November |
| B | Bravo | | O | Oscar |
| C | Charlie | | P | Papa |
| D | Delta | | Q | Quebec |
| E | Echo | | R | Romeo |
| F | Foxtrot | | S | Sierra |
| G | Golf | | T | Tango |
| H | Hotel | | U | Uniform |
| I | India | | V | Victor |
| J | Juliet | | W | Whiskey |
| K | Kilo | | X | X-ray |
| L | Lima | | Y | Yankee |
| M | Mike | | Z | Zulu |

### Usage Example

**To spell "FIRE" over a noisy radio:**
"F... Foxtrot. I... India. R... Romeo. E... Echo. Word is FIRE."

This is much clearer than spelling "F... I... R... E." where the listener might mishear.

### Numbers with Clarity

Spell out numbers if important (radio static can make numbers confusing):
- "Zero" not "O"
- "Niner" not "nine" (nine can sound like five)
- "Two" not "to" or "too"
- "Three" not "tree"
- "Four" not "for"
- "Five" not "five" (can sound like nine with static)

Example: "We have forty-two people" is clearer as "We have four-two personnel" (4-2 clear digits).

## Brevity Codes & Terminology

Pre-established short phrases reduce message length and error.

### Emergency Response Codes

| Code | Meaning | Usage |
|------|---------|-------|
| **STRUCTURE FIRE** | Building actively burning | "STRUCTURE FIRE, Sector Two, evacuate west" |
| **MEDICAL EMERGENCY** | Immediate medical response needed | "MEDICAL EMERGENCY, intersection north road and mill, send 2 personnel" |
| **INTRUDER** | Unauthorized person present | "INTRUDER reported sector south, lock buildings" |
| **ALL CLEAR** | Threat over, area secured | "ALL CLEAR, sector east, resume normal operations" |
| **SHELTER IN PLACE** | Don't move, stay where you are | "SHELTER IN PLACE, toxic cloud approaching from north" |
| **EVACUATE** | Leave immediately to designated area | "EVACUATE west, assembly point is hill north" |

### Direction Codes

Instead of "left" (ambiguous from whose perspective):

- **NORTH, SOUTH, EAST, WEST:** Use compass directions
- **NEAR, FAR:** Close to us vs. distant
- **UP, DOWN:** Higher/lower elevation
- **INBOUND, OUTBOUND:** Toward settlement / away from settlement

Example instead of "go left":
"PROCEED WEST" or "PROCEED NORTH-WEST" is unambiguous.

### Status Codes

| Code | Status |
|------|--------|
| **CODE GREEN** | Normal operations; all systems functional |
| **CODE YELLOW** | Alert; potential threat; elevated readiness |
| **CODE RED** | Active emergency; full response activated |

### Distance Markers

For rough distance communication:

- **NEAR:** < 100 meters / visible clearly
- **CLOSE:** 100-500 meters / can see details with effort
- **FAR:** 0.5-2 kilometers / visible but small
- **DISTANT:** > 2 kilometers / barely visible

Example: "FIRE spotted DISTANT, north-east, appears CLOSE to forest line" = fire is far away but near forest.

## Standardized Message Formats

Structured message formats ensure all critical information is communicated consistently.

### SALUTE Report (Enemy Observation)

When spotting potential threats:

| Letter | What | Example |
|--------|------|---------|
| **S** | Size | "Approximately 20-30 persons" |
| **A** | Activity | "Moving slowly east on main road, appear to be scouts" |
| **L** | Location | "North-east, 2 kilometers, in tree line" |
| **U** | Uniform/Appearance | "Mix of armed and unarmed; some wearing blue cloth" |
| **T** | Time | "Observed 15 minutes ago; may have moved since" |
| **E** | Equipment | "Carrying bows, spears, one mounted on horse" |

**Example SALUTE Report:**
"SALUTE: Size twenty personnel. Activity moving southeast on north road, appear military. Location 1.5 kilometers northeast. Uniform mostly armed with spears and bows. Time observed two-zero minutes ago. Equipment includes one horse-mounted scout. END SALUTE."

### LACE Report (Defensive Position Observation)

When scouting enemy/friendly positions:

| Letter | What | Example |
|--------|------|---------|
| **L** | Location | Where is the position? |
| **A** | Avenues of Approach | How can you reach it? What roads/paths? |
| **C** | Cover & Concealment | What natural shelter? Trees, buildings, terrain? |
| **E** | Equipment | What weapons/defenses? Barriers, trenches? |

### Medical Emergency Format

When reporting medical situation:

"MEDICAL EMERGENCY. [Number] casualties. [Injury types: trauma, burns, poisoning, medical]. Exact location: [specify]. Assistance requested: [ambulance, medical officer, supplies]. Immediate actions taken: [what's being done]. Time from now until critical: [estimated]."

Example:
"MEDICAL EMERGENCY. Two casualties. Burns from fire. Location: sector two, grain warehouse. Assistance: send healer immediately. Actions: cooling with water, movement to open air. Critical in thirty minutes if not treated."

## Radio Voice Procedure

Discipline in radio communication prevents misunderstandings.

### Basic Radio Discipline

**Transmission Structure:**
1. **Call:** Addressee call sign
2. **From:** Your call sign
3. **Message:** The actual information
4. **End:** Confirmation word ("Out")

Example:
"Sector Two, this is Command. Evacuate north immediately. Out."

### Phonetic Transmission

When important information is spelled:

"Spell that name: November-Oscar-Bravo-Echo-Lima. Break. That's a man's name from the north settlement. Out."

(The word "Break" indicates pause in transmitting; allows receiver to ask for repeat if needed.)

### Confirmation Protocol

After receiving important message:

**Receiver responds:** "Command, Sector Two. Message received. Evacuate north immediately. Time zero-six-three-zero hours. Out."

Receiver repeats the message back to confirm understanding.

**Sender confirms:** "Sector Two, that's correct. Out."

### Radio Terminology

| Term | Meaning | Usage |
|------|---------|-------|
| **Roger** | Message received and understood | "Roger, standing by" |
| **Wilco** | Will comply with instruction | "Wilco, heading north now" |
| **Out** | End of transmission; no response expected | "End message. Out." |
| **Over** | End of transmission; response expected | "Standing by for situation update. Over." |
| **Break** | Pause in transmission | "First item. Break. Second item." |
| **Copy** | Confirm understanding | "Copy that, moving to sector two" |
| **Say again** | Repeat last message | "Say again last transmission, signal was weak" |

<section id="relay">

## Relay Message Accuracy

In long chains of communication (A → B → C → D), errors accumulate.

### Relay Discipline

**Receiving the message:**
- Listen fully before responding
- Ask for clarification immediately if unclear
- Repeat the message back to confirm

**Relaying the message:**
- Transmit EXACTLY as received, word-for-word
- Don't paraphrase or interpret
- Use same format (phonetic spelling, terminology)

**Example of poor relay:**
- A to B: "Send 10 fighters north"
- B to C (paraphrasing): "We need reinforcements north"
- C to D (interpreting): "Everyone go north"
- D misunderstands: Sends all remaining fighters, leaving settlement undefended

**Example of good relay:**
- A to B: "Send one-zero fighters north" (phonetically clear)
- B confirms: "Understood. Send ten fighters. North direction. Out."
- B to C: "Send one-zero fighters north" (exact repeat)
- C confirms and relays exactly

### Relay Chain Verification

For critical multi-hop messages:

1. **A** sends to **B**: "Critical message follows. Message: [text]. Acknowledge."
2. **B** repeats back to **A**: "Received. Message: [text]. Out."
3. **B** sends to **C** with same discipline
4. Continue through chain
5. **Final recipient** sends confirmation back through entire chain

This ensures no mutations in the message.

## Shouted Commands in Noise

In fire, combat, or machinery noise, voice communication becomes challenging.

### Penetrating Shout

- **Project from diaphragm:** Deep breath, shout from stomach not throat
- **Sustain vowels:** Don't shout just consonants; draw out the vowel sound
- **Repeat:** Say same command 2-3 times
- **Gesture:** Use hand signals in parallel (point direction while shouting "NORTH")

### High-Priority Commands

In chaos, people listen for high-priority words. Lead with them:

**Poor:** "I think we should probably go to the western side of the settlement because I see smoke in the east."
- Too long, too many words, unclear priority

**Good:** "WEST! Everyone proceed WEST! Smoke in EAST! MOVE WEST NOW!"
- Priorities first, actions clear, repeated

### Pre-Arranged Signals

In predictably noisy situations (combat, loud machinery):

- **One long blast on horn:** Cease fire / stop all work
- **Two short blasts:** Attention / listen up
- **Rapid short blasts:** Emergency / evacuate immediately
- **Continuous blast:** All-clear / return to work

Train everyone on these signals beforehand.

</section>

<section id="hand-signals">

## Hand Signals for Tactical Movement

When silence is required (stealth) or noise is too loud (battle):

### Basic Hand Signals

| Signal | Meaning | How To |
|--------|---------|---------|
| **Fist raised** | Stop / freeze | Raise arm, closed fist |
| **Open palm, sweep down** | Get down / take cover | Open hand, push downward motion |
| **Pointing finger** | Move that direction | Point directly; arm extended |
| **Circular hand motion** | Gather around / regroup | Wave hand in circles above head |
| **Two fingers to eyes, point** | Look / observe that direction | Touch eyes, point in direction |
| **Thumbs up** | Good / proceeding as planned | Raise fist with thumb extended upward |
| **Thumbs down** | Stop / problem detected | Raise fist with thumb extended downward |
| **Index and middle finger pointing** | Two people / pair up | Hold up two fingers |
| **Waving hand low** | Come toward me | Wave hand at waist level |
| **Waving hand high** | Go away / retreat | Wave hand above head |

### Distance Communication with Signals

For long-distance visual signals when voice won't carry:

- **Waving arms:** Large movements visible 100+ meters away
- **Flagging:** Hold cloth/flag, wave in patterns
- **Jumping/movement:** Exaggerated movement captures distant attention
- **Fire/smoke:** Already covered in communication-redundancy-planning guide

<section id="whisper">

## Whisper Networks & Stealth Communication

When speaking aloud risks detection:

### Effective Whisper

- **Breathe deeply first:** Don't run out of air mid-sentence
- **Whisper from diaphragm:** Not sibilant hissing; clear enunciation in low voice
- **Minimal volume:** Low but understandable to immediate neighbor
- **Mouth-to-ear:** Speak directly toward listener's ear (3-6 inches away)

**Example of poor whisper:**
- Hissing "psst, psst, the guards are coming" (too much sibilance, looks suspicious)

**Example of good whisper:**
- Lean in, say clearly but quietly: "The guards are coming. Move to the barn." (clear words, low volume)

### Whisper Chain (Telephone Game)

In large groups, information passes person-to-person via whisper:

**Correct procedure:**
- Speaker whispers message exactly, word-for-word
- Listener repeats back to confirm
- Listener whispers to next person, using exact words
- Continue through group

**Wrong approach (leads to errors):**
- People try to paraphrase in whispers
- Information mutates as it passes
- Final message is corrupted

Train whisper chains beforehand; practice passes incorrect messages through to demonstrate how mutations occur.

</section>

<section id="stress-effects">

## Stress Effects on Communication

Under extreme stress, normal communication breaks down.

### How Stress Affects Speech

| Effect | Manifestation | Mitigation |
|--------|---------------|-----------|
| **Faster speech** | Words run together; harder to understand | Practice speaking slower under stress; force pauses |
| **Higher pitch** | Voice becomes squeaky; less intelligible | Breathe deeply; lower pitch consciously |
| **Reduced vocabulary** | Can't find words; use "um" and "uh" excessively | Pre-learned phrases reduce need for novel word choice |
| **Tunnel vision** | Forget to check understanding; miss contextual info | Use standardized formats to ensure all data points covered |
| **Memory loss** | Forget important details; confusion about sequence | Write down message if time permits; use checklist |
| **Repetition** | Say same thing multiple times without realizing | Use structured format; confirmation procedure prevents repetition |

### Training Under Stress

**Inoculation Training:**

Practice communication in progressively stressful scenarios:

1. **Low stress:** Calm conversation about hypothetical emergency
2. **Moderate stress:** Role-play emergency scenario with timer pressure
3. **High stress:** Live drill with actual sirens, crowding, time pressure, physical exertion
4. **Extreme stress:** Full-scale emergency exercise with multiple simultaneous problems

Practiced skills remain functional under stress; novel communication fails.

</section>

## Communication with Non-Native Speakers

In diverse communities, language barriers are common.

### Accommodation Strategies

**Speech modifications:**
- Speak more slowly (not louder; people aren't deaf)
- Use shorter sentences
- Pause between clauses
- Check for understanding frequently

**Vocabulary choices:**
- Use common words, avoid slang or idioms
- Avoid complex sentences
- Explain technical terms before using them

**Visual aids:**
- Point to objects
- Use hand gestures/signals
- Draw diagrams
- Physical demonstration instead of explanation

### Verification Protocol

For critical messages with non-native speaker:

1. **Speak the message:** "Move north toward the tree line"
2. **Verify understanding:** "Can you show me which direction is north?" (listener points)
3. **Check response:** "Good. Move to the tree line north. Understand?"
4. **Observe action:** Watch that they move in correct direction

Never assume understanding; verify through action.
